AMEVA Scrollbar

CSS-stylable overlay scrollbar

A framework-free overlay scrollbar with DOM-based tracks and thumbs. It can be styled freely with CSS effects such as glow, blur, gradients, shadows and glass-style overlays.

Vertical element scrollbar

Layered UI elements can keep their native scrolling behavior while using a custom visual scrollbar.

The scrollbar is rendered as DOM overlay and can be styled freely with CSS.

This makes glow, blur, shadows and glass effects possible without relying on browser scrollbar styling.
